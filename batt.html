BANTUAN

Mengapa API tidak tersedia?

Chrome Android 80+ menghapus getBattery() untuk privasi.

Browser yang mendukung:

Chrome/Edge Desktop, Opera, Kiwi Browser (Android — direkomendasikan).

Mode Estimasi:

Menggunakan waktu layar aktif, aktivitas CPU (Performance Observer), memori, dan tipe jaringan. Semakin sering log manual, makin akurat.

Tips akurasi:

  • Log setiap 15-30 menit
  • Jaga tab tetap terbuka
  • Aktifkan "Jangan tidurkan layar"
